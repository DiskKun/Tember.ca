Advear

Current development stage: Alpha

Story

In the future, government researchers with the technology allowing them to look into any moment in the past notice an anomaly that occured some odd 500 years ago: for exactly nine seconds, the inhabitants of the country Advear vanished, and then returned. No historical records documented this event, implying that someone (or something!) has been tampering with the past.

You have been assigned the special operatives task of travelling back in time using experimental technology to investigate the incident. There's only one problem: your time machine is only accurate enough to let you visit one year before the incident, and one year after.

The Adverian Language

The written (and spoken!) language of the Adverians will be a fully realized language with a unique alphabet and grammer. I'm developing this language myself, and it will be a major part of the game. The game's manual will teach you how to read the language and translate various texts. In the game, the Adverians do not speak English very well. Some are fluent, some speak it in broken parts, and some will strictly speak Adverian. Knowing the language will be important to the game

About the Game

Advear was originally going to be a browser-based HTML5 game. I decided that I would rather make it in assembly for the Nintendo Game Boy, so that's what I'm doing. In the very early stages of development, the game was open source. However, I have since decided to close the source. Since I plan on releasing the game on a physical Game Boy cartridge, I plan on including the source code in said package.

The game is not finished. The release date is uncertain. I understand that it is an ambitious project. I suspect that it will take me years to complete, as I have to juggle school, work, and other various bothers (:P) in order to find time to work on the game. Anyways, it will release when it's done. Be patient.

Development
